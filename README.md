# 🎓졸업작품 - 눈을 이용한 졸음운전 감지 시스템

### 작품 소개
* 졸음운전은 최근 10년간 발생원인별 고속도로 교통사고율의 1위이며 화물차 교통사고의 원인중 과반을 차지한다.
* 이를 사전에 방지하는것이 목적이다.

### 개발 시 기대효과
* 사고율 감소
* 운전습관 교정

### 개발 기간 : 2021.03 ~ 2021.06
